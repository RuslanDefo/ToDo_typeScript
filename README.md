Hello, this is the worlds best ToDo app using postgreSQL as database. <br>
For use this amazing app follow this instructions:
<hr/>
At first you need to install postgresDB and NODE.js on your PC/server <br>

then run script "npx webpack" to compile this client app. 
<hr/>
run "node ./src/backend/scripts_creators/database.js" for create database, default settings for this database: <br>
  user: 'postgres', <br>
  password: '784944', <br>
  host: 'localhost', <br>
  port: '5432'<br>
<hr/>
run "node ./src/backend/scripts_creators/table.js" for create table for this database
<hr/>
run "nodemon ./src/backend/app.js" for start database
<hr/>
run script "npx webpack serve --mode development --open" to run client app and use this WORLD BEST APPLICATION.
